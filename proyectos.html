Proyectos

A lo largo de mi carrera, he desarrollado diversos proyectos que abarcan diferentes industrias y necesidades empresariales.

Sistema de Biblioteca

Una plataforma completa para la gestión de bibliotecas, que incluye préstamos, devoluciones y administración de catálogos.

Sistema de Notas

Un sistema escolar para la administración de calificaciones, materias y estudiantes, optimizado para profesores y alumnos.

E-commerce de Artesanías

Una tienda en línea para la venta de productos artesanales, con integración de pagos y gestión de inventario en tiempo real.

Sistema de Inventarios

Software diseñado para gestionar eficientemente el stock de productos en bodegas, con alertas y reportes detallados.

Control de Pacientes

Aplicación para hospitales que gestiona historiales médicos, citas y control de pacientes, mejorando la atención sanitaria.

Gestor de Facturas Electrónicas (DTE)

Un sistema para la gestión y emisión de Documentos Tributarios Electrónicos (DTE) para empresas que buscan automatizar su facturación.
